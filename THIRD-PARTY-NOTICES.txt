PdfSharp 6.2.4
License: MIT
Source: https://github.com/empira/PDFsharp
────────────────────────────────────────

Middleman.PdfFlex includes a modified fork of PDFsharp 6.2.x (MIT License,
copyright empira Software GmbH). The fork covers the PDF object model,
drawing primitives (XGraphics, XFont, XImage), PDF reader/writer, security,
digital signatures, and universal accessibility modules. MigraDoc and
platform-specific builds (GDI+/WPF) are not included. Modifications include
a PDF conformance system (PDF/A, PDF/UA, PDF Declarations), watermark
rendering fixes, streaming output support, and namespace consolidation
under Middleman.PdfFlex.

────────────────────────────────────────
StbImageSharp 2.30.15
License: Public Domain (Unlicense) / MIT dual-licensed
Source: https://github.com/StbSharp/StbImageSharp
────────────────────────────────────────

StbImageSharp (a pure C# port of stb_image.h) source is embedded directly
for PNG, BMP, GIF, TGA, PSD, and HDR image decoding with zero external
dependencies. Compiled with STBSHARP_INTERNAL so all types are internal.
JPEG files are handled via passthrough (not decoded by StbImageSharp).

The Noto font family by Google is bundled as embedded resources for universal
language and script coverage as a fallback when system fonts are unavailable.
